FLLTutorials Tournament Rubrics

We provide two options for using our scorer in a tournament:

  • Manage yourself: Use the modified Core Values, Innovation Project, and Robot Design rubrics that allow you to calculate a score for each team, then download a single file with a unique name. You can then load that file back into the system to view/edit the scores. You manage the files yourself on your hard drive, Google Drive, etc.
  • Full tournament system: Use this complete tournament scoring system that will manage and save referee scores and rubrics on the FLLTutorials server. It will generate entries for each team in your tournament and display them onto a scoreboard.
